# **App Name**: Profolio Pakistan

## Core Features:

- Interactive CV Builder: A step-by-step form to input personal info, education, experience, skills, and references.
- ATS-Friendly CV Generation: Generates CVs that are parsed correctly by Applicant Tracking Systems. Incorporates a tool which assesses each CV's formatting for ATS compatibility.
- Template Selection: Offers a range of pre-designed CV and portfolio templates.
- PDF Export: Allows users to download their CVs and portfolios as PDF files.
- Portfolio Builder: Enables users to create personal portfolios with projects, images, links, and testimonials.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for a professional, trustworthy feel. The choice of dark blue conveys stability and sophistication, appealing to the target audience.
- Background color: Very light blue (#DDE6ED), near-white, providing a clean and modern backdrop.
- Accent color: Soft violet (#5C6B8A), for highlighting calls to action.
- Body and headline font: 'Inter', a sans-serif font, for a modern and clean aesthetic.
- Use clean and professional icons that align with the text and enhance the visual appeal.
- Mobile-first approach with a clean, minimal, and professional design for easy navigation.
- Subtle animations and transitions for a smooth user experience.
